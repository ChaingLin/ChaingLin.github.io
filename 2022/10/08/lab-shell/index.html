实验一 使用shell

一、实验目的

  1. 熟悉 shell 常用命令

  2. 熟悉 git 常用命令

  3. 熟悉 gitee 如何使用

二、自测题

  1. Linux 中创建文件夹的命令是

    mkdir

  2. Linux 中直接进入根目录的命令是

    cd

  3. 在使用 chmod 命令时，chmod +755 是什么含义?

    • 文件所有者可读可写可执行

    • 与文件所有者同属一个用户组的其他用户可读可执行

    • 其它用户组可读可执行

  4. 查找当前目录及其子目录下所有文件后缀为 .c 的文件的命令是

    find . -name '*.c' -print

  5. 将 文 件 名 为 hello 改 为 world 的 命 令 是

    mv hello world

  6. 查看当前路径的命令是

    pwd

  7. 在 Linux 终端中输入 echo date 后会有什么样的结果?

    返回date

  8. 想要把字符串”hello”追加到当前目录下的 test.txt 文件，我应该怎样输

    入命令?

    echo "hello">> test.txt

  9. 在 shell 脚 本 中 #! 是 什 么 意思?

    \#!是特殊的表示符，其后面根的是此解释此脚本的shell的路径。

  10. shell 脚本中$?是什么意思?

    显示最后命令的退出状态。0表示没有错误，其他任何值表明有错误。

  11. 我想查找历史中使用过的 ls 命令，我应该输入什么样的命令？

    cat ~/.zsh_history | grep ls

    cat 查看shell历史记录文件，并用grep命令查找出历史的ls

  12. 在 Linux 中 ， 挂 载 应 该 使 用 哪 个 命令?

    mount

    3. 用 touch 在 ouc 文件夹中新建一个叫 soft 的文件。
    4. 将以下内容一行一行地写入 soft 文件:
      #!/bin/sh
      curl –head –silent http://it.ouc.edu.cn
      # 在 Bash 中表示注释，而 ! 即使有双引号(“)也有特殊含义。单引号(‘)……
    5. 直接执行 soft。如果程序无法执行，使用 ls 命令来获取信息并理解原因。
    6. 查看 chmod 的手册(可使用 man chmod)
    7. 使用 chmod 命令改变权限，使能够成功执行。不要用 sh soft 。

    思考: shell 为何知道用 sh 来解析这个文件?
    使用| 和 > ，将 soft 文件的最后更改日期信息，写入主目录下 la.txt。
